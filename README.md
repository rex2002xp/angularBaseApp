# Angular Base App

Este proyecto fue generado con [yo angular generator](https://github.com/yeoman/generator-angular)
versión 0.15.1.

Para utilizarlo pueden referirse a la documentacion de [yo angular generator](https://github.com/yeoman/generator-angular) considerando que unos pocos paquetes han sido reemplazados por los que considero que son mas convenientes para el desarrollo de aplicaciones con Angular.

### Paquetes incorporados.

El proyecto es una variante en la cual he incorporado algunos paquetes que son de mi gusto personal.

Los paquetes agregados son:
- grunt-browser-sync
- grunt-contrib-less
- angular-ui-router

Paquetes retirados :
- grunt-contrib-connect
- angular-route

Como puede verse el cambio es mínimo, pero deseo que sean de su agrado para inspirar nuevas aplicaciones Angular que mejoren la experiencia de sus usuarios.

### Instalación

La instalación es sencilla :

```sh
$ git clone https://github.com/rex2002xp/angularBaseApp.git angularBaseApp
$ cd angularBaseApp
$ npm install
$ bower update
```

## Generación de Build

```sh
$ grunt
```

### Ejecutar la aplicación en desarrollo

```sh
$ grunt serve
```
